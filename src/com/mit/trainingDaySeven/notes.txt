Checked and Unchecked Exceptions:
* Checked - Compile Time Exceptions, may or may not occur but should add exception handling
* Unchecked - Runtime Exceptions, may or may not occur but no need to add exception handling

Exception Examples:
* ArithmeticException -> Divide By Zero
* ArrayIndexOutOfBoundsException -> Array Index Out Of Bound
* NullPointerException -> Null Value
* IOException -> Input Output Exception
* FileNotFoundException -> File Not Found
